# [Terminal Portfolio Website](https://www.maschad.codes/)


## Features
* **[Tab]** for auto completion.
* **[Esc]** to clear the input line.
* **[↑][↓]** to scroll through your command history.


## Run the Project Locally:
```
Install the dependencies
```shell
npm install
```
Start the server
```shell
npm run dev
```
